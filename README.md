# Jeu de Tir Spatial avec Pygame

Un jeu de tir spatial simple réalisé en utilisant la bibliothèque Pygame de Python. Dans ce jeu, le joueur contrôle un vaisseau spatial pour détruire des ennemis qui apparaissent à l'écran.

## Prérequis:

Avant de lancer le jeu, assurez-vous d'avoir Python installé sur votre ordinateur. Vous pouvez installer Pygame en utilisant pip :
```bash
pip install pygame
```

## Comment jouer:

Utilisez les touches fléchées gauche/droite pour déplacer le vaisseau spatial.
Appuyez sur la barre d'espace pour tirer des projectiles.
Détruisez autant d'ennemis que possible tout en évitant les collisions avec eux.
